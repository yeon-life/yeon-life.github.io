연의 사랑방 위젯 — 설치 테스트 페이지

✅ 위젯이 제대로 작동하면
오른쪽 아래에 검은색 "緣에 묻다" 플로팅 버튼이 보이고,
클릭하면 다섯 겹 응답이 펼쳐지는 모달이 열립니다.

이 페이지는 사랑방 위젯이 어떤 호스트 페이지(연에이젼트, 연플래너, 수풀AI 등 어떤 프로젝트든)에 설치해도 똑같이 작동하는지 확인하기 위한 샘플 페이지입니다.

설치 방법

HTML의 </body> 직전에 한 줄만 넣으면 끝:

<script src="saranbang-widget.js"></script>

선택 옵션 (필요할 때만)

<script>
  window.YeonSaranbangConfig = {
    lang: 'ko',                 // 'ko' 또는 'en' (기본: 자동 감지)
    position: 'bottom-right',   // 'bottom-right' 또는 'bottom-left'
    host: 'Yeon Agent'          // 위젯이 어디에 설치됐는지 표시용
  };
</script>
<script src="saranbang-widget.js"></script>

JavaScript API

YeonSaranbang.open();        // 수동으로 모달 열기
YeonSaranbang.close();       // 수동으로 닫기
YeonSaranbang.setLang('en'); // 언어 전환
YeonSaranbang.version;       // 현재 버전 확인

특징

  • 단일 파일 — 외부 의존성 없음 (Google Fonts만 자동 로드)
  • CSS 스코핑 — 모든 클래스에 ys- 접두어. 호스트 페이지 스타일과 충돌 없음
  • 접근성 — ESC 키로 닫기, aria 속성 포함
  • 반응형 — 모바일·태블릿·PC 자동 대응
  • 언어 자동 감지 — <html lang> 또는 브라우저 언어 기반

설치 위치

  • Y Life 홈페이지 — 이미 설치됨 (index.html 등 모든 페이지)
  • 연에이젼트 — 설치 예정
  • 연플래너 / 수풀AI / 스무고개 — 필요 시 추가 가능
